# מבוא למדעי המחשב - סיכום

## משתנים (Variables)

משתנה הוא מקום בזיכרון המחשב שמאחסן ערך מסוים.

### סוגי משתנים בסיסיים:
- **int** - מספרים שלמים (לדוגמה: `5`, `-3`, `0`)
- **float** - מספרים עשרוניים (לדוגמה: `3.14`, `-2.5`)
- **str** - מחרוזות טקסט (לדוגמה: `"שלום"`, `"hello"`)
- **bool** - ערכים בוליאניים (`True` או `False`)

### דוגמאות:
```python
age = 25          # מספר שלם
price = 19.99     # מספר עשרוני
name = "דני"      # מחרוזת
is_student = True # בוליאני
```

## תנאים (Conditions)

תנאים מאפשרים לתוכנית לבצע פעולות שונות בהתאם למצבים שונים.

### מבנה בסיסי:
```python
if תנאי:
    # קוד שירוץ אם התנאי מתקיים
elif תנאי_אחר:
    # קוד שירוץ אם התנאי האחר מתקיים
else:
    # קוד שירוץ אם אף תנאי לא מתקיים
```

### אופרטורים להשוואה:
| אופרטור | משמעות |
|---------|--------|
| `==` | שווה |
| `!=` | שונה |
| `>` | גדול מ |
| `<` | קטן מ |
| `>=` | גדול או שווה |
| `<=` | קטן או שווה |

### אופרטורים לוגיים:
- **and** - וגם (שני התנאים חייבים להתקיים)
- **or** - או (לפחות תנאי אחד חייב להתקיים)
- **not** - שלילה (הופך את התנאי)

```python
age = 20
has_license = True

if age >= 18 and has_license:
    print("יכול לנהוג")

if age < 13 or age > 65:
    print("הנחה בכרטיס")

if not has_license:
    print("צריך רישיון")
```

### דוגמה:
```python
grade = 85

if grade >= 90:
    print("מצוין!")
elif grade >= 70:
    print("טוב מאוד")
else:
    print("יש מקום לשיפור")
```

## לולאות (Loops)

לולאות מאפשרות לחזור על פעולה מספר פעמים.

### לולאת while
רצה כל עוד התנאי מתקיים:
```python
count = 0
while count < 5:
    print(count)
    count += 1
# ידפיס: 0, 1, 2, 3, 4
```

### לולאת for
רצה על כל איבר באוסף או בטווח מספרים:
```python
# מעבר על טווח מספרים
for i in range(5):
    print(i)
# ידפיס: 0, 1, 2, 3, 4

# מעבר על רשימה
fruits = ["תפוח", "בננה", "תפוז"]
for fruit in fruits:
    print(fruit)
```

### פקודות שליטה בלולאה:
- **break** - יציאה מיידית מהלולאה
- **continue** - דילוג לאיטרציה הבאה

### דוגמה:
```python
for i in range(10):
    if i == 3:
        continue  # דלג על 3
    if i == 7:
        break     # עצור ב-7
    print(i)
# ידפיס: 0, 1, 2, 4, 5, 6
```

## טיפים חשובים

1. **הזחה (Indentation)** - בפייתון, הזחה קובעת את מבנה הקוד. השתמשו ב-4 רווחים או Tab באופן עקבי.

2. **שמות משתנים** - בחרו שמות משמעותיים שמתארים את תוכן המשתנה.

3. **תנאים מקוננים** - ניתן לשים תנאי בתוך תנאי, אך יש להיזהר מקינון עמוק מדי.

4. **לולאות אינסופיות** - יש לוודא שתנאי העצירה בלולאת while יתקיים בשלב כלשהו.

| טעות | דוגמה שגויה | תיקון |

| שכחת נקודתיים | `if x > 0` | `if x > 0:` |
| הזחה לא עקבית | ערבוב Tab ורווחים | להשתמש רק באחד מהם |
