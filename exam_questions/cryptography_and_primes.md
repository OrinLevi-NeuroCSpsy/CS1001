## שאלה 2: modpower - ריבוע חוזר

```python
def modpower(a, b, c):
    result = 1
    while b > 0:
        if b % 2 == 1:
            result = (result * a) % c
        a = (a * a) % c
        b = b // 2
    return result
```

**סיבוכיות:** O(log b) - מספר הביטים של b

**דוגמה:** חשב 5^100 mod 7
- לפי פרמה: 5^6 ≡ 1 (mod 7)
- 100 = 6×16 + 4
- 5^100 ≡ 5^4 = 625 = 89×7 + 2 ≡ 2 (mod 7)

## שאלה 3: יחס בין מספרים בייצוג בינארי (2023b מועד א', שאלה 1א')

**נתון:** a בעל 2n ביטים, b בעל n ביטים

**שאלה:** מהם הגבולות של a/b?

**פתרון:**
- a_min = 2^(2n-1) (מספר מינימלי עם 2n ביטים)
- a_max = 2^(2n) - 1
- b_min = 2^(n-1)
- b_max = 2^n - 1

**חסם תחתון:** a_min / b_max = 2^(2n-1) / (2^n - 1) ≈ 2^(n-1)

**חסם עליון:** a_max / b_min = (2^(2n) - 1) / 2^(n-1) < 2^(n+1)
