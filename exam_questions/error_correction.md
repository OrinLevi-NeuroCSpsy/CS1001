## שאלה 1: אורך ומרחק קוד (2022b מועד א', שאלה 1ה')

**נתון:**
```python
def code(w):
    k = len(w)
    return w + "".join([w[i]*3 for i in range(0, k, 2)])
```

**שאלה:** מהו אורך מילת הקוד n ומרחק הקוד d?

**פתרון:**
- אורך הודעה: k (זוגי)
- מוסיפים: k/2 * 3 = 3k/2 ביטים
- **n = k + 3k/2 = 5k/2**

- מרחק: צריך למצוא שתי הודעות עם מרחק מקסימלי
- **d = 4** (לכל ביט במיקום זוגי, יש 4 עותקים)

## שאלה 2: XOR-based קוד (2023a מועד א', שאלה 1ה')

**נתון:**
```python
def C(w):
    k = len(w)
    res = [x for x in w]
    for i in range(k):
        for j in range(i+1, k):
            bit = str((int(w[i]) + int(w[j])) % 2)
            res.append(bit)
    return "".join(res)
```

**שאלה:** מהו אורך מילת קוד ומרחק הקוד עבור k=5?

**פתרון:**
- אורך: k + C(k,2) = k + k(k-1)/2 = **k(k+1)/2**
- עבור k=5: 5 + 10 = 15

## שאלה 3: הוכחת קיום מילה במרחק (2023b מועד א', שאלה 1ו')

**נתון:** קוד C: {0,1}² → {0,1}⁸ עם d=2 בין כל שתי מילות קוד.

**שאלה:** הוכח שקיימת מילה z ∈ {0,1}⁸ שאינה מילת קוד ומרחקה מכל מילת קוד לפחות 2.

**הוכחה:**
- יש 4 הודעות → 4 מילות קוד
- כדורי המינג ברדיוס 1 מכילים 1+8=9 מילים כל אחד
- אבל הכדורים לא בהכרח זרים (d=2)
- סה"כ מילים: 2⁸ = 256
- 4 כדורים × 9 = 36 מילים לכל היותר
- 256 - 36 = 220 מילים שלא בכדור כלשהו

## שאלה 4: טבלת סיכום - יכולות קוד

| מרחק d | גילוי | תיקון |
|--------|-------|-------|
| 2 | 1 שגיאה | - |
| 3 | 2 שגיאות | 1 שגיאה |
| 4 | 3 שגיאות | 1 שגיאה |
| 5 | 4 שגיאות | 2 שגיאות |

**נוסחה:**
- גילוי: d-1 שגיאות
- תיקון: ⌊(d-1)/2⌋ שגיאות
